# Finly — Dashboard Fintech

Projeto desenvolvido para a disciplina de Análise e Desenvolvimento de Sistemas.
Tela de dashboard de um aplicativo financeiro fictício chamado **Finly**.

---

## Como abrir o projeto

1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` diretamente no navegador
3. Não é necessário instalar nada

---

## Estrutura de arquivos

```
finly-fintech/
├── index.html          (página principal)
├── style/
│   └── styles.css      (estilos customizados)
├── images/
│   └── dashboard-finly.png
└── README.md
```

---

## Tecnologias utilizadas

- **HTML5** — estrutura da página
- **CSS3** — estilos customizados (separado do HTML)
- **Tailwind CSS** — framework de estilização via CDN
- **Google Fonts** — fontes Poppins e Montserrat

---

## O que a tela apresenta

- Saldo total disponível com variação mensal
- Resumo de receita, despesa e saldo líquido
- Gráfico de gastos semanais feito com CSS puro
- Lista de movimentações recentes com status
- Card de transferência rápida
- Visualização de cartão de crédito
- Barra de navegação inferior para dispositivos móveis

---

## Responsividade

- **Desktop:** sidebar lateral fixa + grade de 3 colunas
- **Mobile:** sidebar oculta + navegação inferior + layout em coluna única
